Document Object Model (DOM). DOM is a system through which html documents All its elements are easily accessible and its main purpose is to modify all elements of html in document.

Here's how to select HTML elements using the DOM.
document.getElementById('')
document.getElementsByClassName('')
document.getElementsByTagName('')
document.querySelector('')
document.querySelectorAll('')
